# Pengguna Helpdesk

## Login

> Merupakan halaman untuk melakukan proses login aplikasi SIWAK, pengguna dapat melakukan Langkah-langkah sebagai berikut:

Buka Halaman Login.

![](https://apps.syscloud.my.id/docs_app/images/1739742484.png)

Klik “Login Pegawai”, aplikasi akan menampilkan form login untuk pegawai.

![](https://apps.syscloud.my.id/docs_app/images/1739742499.png)

Masukkan NIK dan kata sandi, kemudian klik tombol “MASUK”.

## Bantuan

### Manage Blangko

> Merupakan menu yang manampilkan permohonan pelepasan blangko rusak menjadi tidak rusak dan pelaporan blangko rusak.

> ![](https://apps.syscloud.my.id/docs_app/images/1739742914.png)

Menyetujui atau menolak pelepasan blangko

Klik pelepasan blangko menjadi tidak rusak

Klik panah biru pada kolom berkas untuk melihat bukti blangko yang dilaporkan

Klik Menu validasi pelepasan blangko yang ada pada kolom nomor blangko

Klik y jika menyetujui pelepasan dan klik tidak jika ingin menolak pelepasan

Isi kolom respon

Klik simpan

![](https://apps.syscloud.my.id/docs_app/images/1739742932.png)

Menyetujui atau menolak pelaporan blangko rusak

Klik menu pelaporan blangko rusak

Klik panah biru pada kolom berkas untuk melihat bukti blangko yang dilaporkan

Klik Menu validasi pelaporan blangko yang ada pada kolom nomor blangko

Klik ya jika menyetujui pelaporsan klik tidak jika ingin menolak pelaporan

Isi kolom respon

Klik simpan

![](https://apps.syscloud.my.id/docs_app/images/1739742950.png)

### Tiket Bantuan

> Merupakan halaman untuk melihat data tiket aplikasi SIWAK, pengguna dapat melakukan Langkah-langkah sebagai berikut:

Pastikan pengguna sudah melakukan proses login pengguna pegawai.

Klik menu “Tiket” pada sidebar.

![](https://apps.syscloud.my.id/docs_app/images/1739742965.png)

Aplikasi akan menampilkan menu tiket.

![](https://apps.syscloud.my.id/docs_app/images/1739742981.png)

Isi form pencarian data permohonan tiket, kemudian klik tombol untuk mencari data tiket.

Klik salah satu tombol status untuk memfilter data tiket berdasarkan statusnya.

![](https://apps.syscloud.my.id/docs_app/images/1739743000.png)
